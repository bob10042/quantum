## 🚀 Quick Start

### View Documentation
**Live Site:** https://bob10042.github.io/quantum/

### Clone Repository
```bash
git clone https://github.com/bob10042/quantum.git
cd quantum
```

### Explore Papers
```bash
# View consciousness mathematics
cd Theory_Papers/Consciousness_Mathematics
code CONSCIOUSNESS_MATHEMATICS.md

# View unified field theory
cd ../Unified_Field_Theory
code Unification_of_Scale_Invariant_and_Consciousness_Model.md
```
